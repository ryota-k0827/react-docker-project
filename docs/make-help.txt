<makeコマンドのヘルプ>

make init
-> 初期化（envファイルの作成など）


<基本コマンド>
make build
-> サービスの構築・再構築

make up
-> コンテナを作成し、開始する

make up-d
-> バックグラウンドでコンテナを作成し、開始する

make down
-> コンテナの停止と削除


<コンテナ内でのコマンド>
make y/install
-> コンテナ内でパッケージのインストールし、ローカルにコピー

make y/add p=<パッケージ名>
-> コンテナ内でパッケージの追加し、ローカルにコピー

make y/add-D p=<パッケージ名>
-> コンテナ内でパッケージを開発環境に追加し、ローカルにコピー

make y/remove p=<パッケージ名>
-> コンテナ内でパッケージを削除し、ローカルにコピー

make y/cache/clean
-> コンテナ内でパッケージキャッシュを全て削除し、クリーンなnode_modulesをローカルにコピー

make y/outdated
-> 古いパッケージ一覧の表示

make y/upgrade p=<パッケージ名>
-> コンテナ内でパッケージを更新し、ローカルにコピー

make y/info p=<パッケージ名>
-> パッケージ情報の表示


<ローカルでのコマンド>
make modules/cp
-> コンテナ内のnode_modulesをローカルにコピー

make rm/modules
-> ローカルのnode_modulesを削除

make rm/build
-> ローカルのビルドファイルを削除

make open/web
-> ブラウザでhttp://localhost:3000を開く


<その他>
make help
-> ヘルプの表示
